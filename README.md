# TempFC
# Temperature Conversion Website

This is a simple temperature conversion website built using HTML, CSS, and JavaScript.

## Description
This website allows users to enter a temperature value and convert it between Celsius and Fahrenheit. The user can choose the type of conversion using radio buttons and click the **Convert** button to see the result instantly.

## Features
- Enter any temperature value.
- Convert **Celsius to Fahrenheit**.
- Convert **Fahrenheit to Celsius**.
- Simple and clean user interface.
- Instant result display after clicking the convert button.

## How It Works
1. The user enters a temperature value in the input box.
2. The user selects the conversion type:
   - Celsius → Fahrenheit  
   - Fahrenheit → Celsius
3. After clicking the **Convert** button, the website calculates the result and displays the converted temperature.

## Technologies Used
- HTML
- CSS
- JavaScript

## Example
If the user enters **8** and selects **Fahrenheit → Celsius**, the result will be:
-13.33 °C

## Purpose
This project was created to practice basic web development and understand how temperature conversion works using JavaScript.
